### Abstract
<div align="justify">
This was a project I did for my Bachelor Thesis, along with Summer Yiu and Bryant Tang.
It is a project based upon the state-of-the-art <a href="https://github.com/AlexeyAB/darknet#how-to-train-tiny-yolo-to-detect-your-custom-objects">YOLO series, Tiny</a> and the famous Kalman Filter. With both camera and object moving, our system is able to track the target robustly in a 3D world. The UAV would then try to maneuver with the dynamic object once it detects the movement from the target. Feel free to check out our code on github!
</div>
