### Abstract
<div align="justify">
This was a project I did for fun back in my Bachlor studies,
where I utilize a two-wheel robot in Gazebo + ROS to execute the
very very very ancient Bug path planner.
Basically, the robot, equipped with 2D laser sensor,
will try to reach the destination via following the "wall" of obstacles.
It is a rather simple project, yet this is the start of my roboticist career!
Feel free to check out the code on github!
</div>
